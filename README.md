# databus-testing

Python script for sending test messages to the databus endpoint.

## Requirements

- Python 3.11+ 
- Packages: `requests` and `PyJWT`

## Setup

```bash
pip install requests PyJWT
```

## Usage

```bash
python3 send-message-to-databus.py
```

## Expected Output

```
SUCCESS! (HTTP 200)
Databus accepted your message.
```
